Laravel Product CRUD Assessment:
This repository contains a Product CRUD developed in Laravel as part of a technical assessment. The application demonstrates clean CRUD operations, image handling, and AJAX-based interactions.

Project Overview:

1. The system allows users to:
2. Create, view, update, and delete products
3. Upload multiple product images
4. Replace old images on update
5. Perform actions without page reload using AJAX

Tech Stack:

1. Laravel 12
2. PHP 8.x
3. MySQL
4. jQuery & AJAX
5. Bootstrap 5
6. Toastr Notifications

Installation & Setup:

1. git clone <github-repository-url>

3. composer install
